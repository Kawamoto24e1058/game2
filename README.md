# オンラインバトルカードゲーム 🎮

言葉を入力してAIに読み込ませ、技を生成してオンラインで戦うバトルカードゲーム！

## 機能

- **AI技生成**: 入力した言葉からAIが強力な技を自動生成
- **多彩な属性**: 炎、水、雷、回復、物理、魔法などの属性
- **オンラインバトル**: リアルタイムで他のプレイヤーと対戦
- **ターン制バトル**: 戦略的なターン制バトルシステム
- **特殊効果**: クリティカル、貫通、火傷、麻痺などの特殊効果

## インストール

```bash
npm install
```

## 実行方法

### ローカル実行

```bash
npm start
```

サーバーが起動したら、ブラウザで `http://localhost:3000` を開いてください。

### オンライン公開（URLで共有）

このゲームを他の人と共有するために、以下の無料ホスティングサービスにデプロイできます：

#### 🚀 Render (推奨)

1. [Render](https://render.com) にアカウント作成
2. "New +" → "Web Service" を選択
3. このGitHubリポジトリを接続
4. 自動的に `render.yaml` の設定が読み込まれます
5. "Create Web Service" をクリック
6. デプロイ完了後、公開URLが発行されます（例: `https://game2-battle-card.onrender.com`）

#### 🚄 Railway

1. [Railway](https://railway.app) にアカウント作成
2. "New Project" → "Deploy from GitHub repo" を選択
3. このリポジトリを選択
4. 自動的にデプロイされ、公開URLが発行されます

#### ⚡ Vercel

1. [Vercel](https://vercel.com) にアカウント作成
2. "Add New Project" でこのリポジトリをインポート
3. 自動的にデプロイされます
4. **注意**: VercelはWebSocketの制限があるため、マルチプレイヤー機能は制限される可能性があります

**推奨**: RenderまたはRailwayを使用してください。両方ともWebSocketを完全にサポートしています。

📖 **詳細なデプロイ手順**: [DEPLOYMENT.md](./DEPLOYMENT.md) を参照してください。

## 遊び方

1. **技を生成**: 言葉を入力して「技を生成」ボタンをクリック
2. **3つの技を作成**: 合計3つの技を生成（最大3つまで）
3. **バトル開始**: 「オンラインバトル開始」ボタンをクリック
4. **対戦相手を待つ**: 他のプレイヤーがマッチングされるまで待機
5. **バトル**: ターン制で技を選択して相手と戦う
6. **勝利条件**: 相手の体力を0にしたら勝利！

## 技の生成ルール

- **言葉の長さ**: 長い言葉ほど強力な技になる
- **特定のキーワード**: 
  - 「火」「炎」「燃える」→ 火属性（火傷効果）
  - 「水」「氷」「凍る」→ 水属性（凍結効果）
  - 「雷」「電気」「ショック」→ 雷属性（麻痺効果）
  - 「癒し」「回復」→ 回復属性（体力回復）
- **文字の構成**: 子音が多い→物理属性、母音が多い→魔法属性
