# オンラインバトルカードゲーム 🎮

言葉を入力してAIに読み込ませ、技を生成してオンラインで戦うバトルカードゲーム！

## 機能

- **AI技生成**: 入力した言葉からAIが強力な技を自動生成
- **多彩な属性**: 炎、水、雷、回復、物理、魔法などの属性
- **オンラインバトル**: リアルタイムで他のプレイヤーと対戦
- **ターン制バトル**: 戦略的なターン制バトルシステム
- **特殊効果**: クリティカル、貫通、火傷、麻痺などの特殊効果

## インストール

```bash
npm install
```

## 実行方法

```bash
npm start
```

サーバーが起動したら、ブラウザで `http://localhost:3000` を開いてください。

## 遊び方

1. **技を生成**: 言葉を入力して「技を生成」ボタンをクリック
2. **3つの技を作成**: 合計3つの技を生成（最大3つまで）
3. **バトル開始**: 「オンラインバトル開始」ボタンをクリック
4. **対戦相手を待つ**: 他のプレイヤーがマッチングされるまで待機
5. **バトル**: ターン制で技を選択して相手と戦う
6. **勝利条件**: 相手の体力を0にしたら勝利！

## 技の生成ルール

- **言葉の長さ**: 長い言葉ほど強力な技になる
- **特定のキーワード**: 
  - 「火」「炎」「燃える」→ 火属性（火傷効果）
  - 「水」「氷」「凍る」→ 水属性（凍結効果）
  - 「雷」「電気」「ショック」→ 雷属性（麻痺効果）
  - 「癒し」「回復」→ 回復属性（体力回復）
- **文字の構成**: 子音が多い→物理属性、母音が多い→魔法属性

## 技術スタック

- **Backend**: Node.js, Express, WebSocket (ws)
- **Frontend**: HTML, CSS, JavaScript
- **リアルタイム通信**: WebSocket
